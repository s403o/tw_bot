# Twitter Bot
Just a simple twitter bot that's taking an images from source and tweet it!.

---

## install
    make setup
    make install
    
---

## Run
### 1. add url of source you want to get images from (.jpg, .png) and run it.
    python source.py    
### 2. init the api auth then run it, it 'll iterates images from folder created by first script :)
    python bot.py
